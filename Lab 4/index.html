Preguntas

  1. ¿Qué diferencias y semejanzas hay entre Java y JavaScript?

    • Semejanzas:

      Los dos se pueden usar para orintado a objetos.

      Los dos tienen sitaxis parecida.

    • Diferencias:

      JavaScript es recomendado para aplicaciones y sitios web (front-end).

      Java es un lenguaje de back-end, destinado a aplicaciones independientes (escritorio y Android).

  2. ¿Qué métodos tiene el objeto Date?

    • getDate(): Devuelve el día del mes.
    • getDay(): Devuelve el día de la semana.
    • getHours(): Retorna la hora.
    • getMinutes(): Devuelve los minutos.
    • getMonth():Devuelve el mes (atención al mes que empieza por 0).
    • getSeconds(): Devuelve los segundos.
    • getTime(): Devuelve los milisegundos transcurridos entre el día 1 de enero de 1970 y la fecha correspondiente al objeto al que se le pasa el mensaje.
    • getYear():Retorna el año, al que se le ha restado 1900. Por ejemplo, para el 1995 retorna 95, para el 2005 retorna 105. Este método está obsoleto en Netscape a partir de la versión 1.3 de Javascript y ahora se utiliza getFullYear().
    • getFullYear(): Retorna el año con todos los dígitos. Usar este método para estar seguros de que funcionará todo bien en fechas posteriores al año 2000.

  3. ¿Qué métodos tienen los arreglos?

    • length: Concoer el tamaño
    • toString(): el array se convierte en una cadena
    • shift(): obtener el primer elemento de un array
    • reverse(): invierte sus elementos
    • sort():ordenar alfabéticamente

  4. ¿Cómo se declara una variable con alcance local dentro de una función?

    Se usa la palabra var seguido del nombre de la variable.Ahora también se usa let para declarar variables limitando su alcance.

  5. ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

    Porque puede ser usada en cualquier punto del código, puede llegar a ser muy difícil medirlo.

Preguntas

  • ¿Por qué es una buena práctica separar el modelo del controlador?

    Para ayudarnos a crear aplicaciones con mayor calidad, tener mejor estructuración de los programas o reutilización del código, lo que debe influir positivamente en la facilidad de desarrollo y el mantenimiento.

  • ¿Qué es SQL injection y cómo se puede prevenir?

    Permite al atacante enviar o “inyectar” instrucciones SQL dentro del código SQL programado para la manipular la base de datos para acceder a algun dato programado

Cuestionario BD

Preguntas:

  1. ¿Qué es entonces una base de datos - DB?

    Un lugar donde se nos permite guardar un conjunto de datos/archivos que estan relacionados entre sí de manera organizada para hacer su búsqueda y utilización más eficiente.

  2. ¿En qué casos es conveniente usar bases de datos?

    En todos los casos donde se necesite almacenar mucha información, por ejemplo:

    • Hay archivos interrelacionados
    • Busquedas dinámicas de información
    • Volumen extenso de datos

  3. ¿Qué es un sistema de gestión de base de datos - DBMS?

    El programa de software que requiere la basee de datos.Sirve como una interfaz entre la base de datos y sus usuarios o programas finales, lo que permite a los usuarios recuperar, actualizar y administrar cómo se organiza y optimiza la información.

  4. Enlista y explica las funciones/responsabilidades que tienen los DBMS.

    • Definición de los datos: definir los objetos de la base de datos
    • Seguridad e integridad de los datos: No todos deben tener acceso a toda la información, restricciones de consistencia
    • Recuperación y restauración de los datos: en caso de fallo poder recuperar los datos sin problema.
    • Control de concurrencia: si hay varias versiones, que haya una que sea consistente con los datos
    • Manipulación de los datos: responder a las solicitudes del usuario de supresión, actualización y extracción, entre otras.

  5. Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.
